# SQLITE Subpackage

A basic wrapper for sqlite database, since the `database/sql` package, while powerful, has a lot in it.
If you want further configuration, go to `mattn/go-sqlite3` for more info on dealing with SQLITE,
otherwise there exist various packages for various databases. 
This library serves as a collection of tools to simplify or centralise frequently done file operations.
